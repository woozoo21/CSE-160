Controls:
Drag mouse → rotate rat
⇧ Shift+Click → poke animation (jump + barrel roll!)
Sliders:
• Camera Angle → rotate view
• Front Leg → joint 1 (upper front legs)
• Back Leg → joint 2 (upper back legs)
• Tail Base → joint 3 level 1 (whole tail)
• Tail Mid → joint 3 level 2 (mid + tip)
• Tail Tip → joint 3 level 3 (tip only)
Animation:
• Run → legs + tail animate fast
• Crawl → body sway animation
• Stop → freeze
