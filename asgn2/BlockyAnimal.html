Controls:
Drag mouse → rotate rat (360 view)
⇧ Shift+Click → poke animation (jump)
Sliders:
1. Camera Angle → rotate view
2. Front Leg → joint 1 (upper front legs)
3. Back Leg → joint 2 (upper back legs)
4. Tail Base → joint 3 level 1 (whole tail)
5. Tail Mid → joint 3 level 2 (mid + tip)
6. Tail Tip → joint 3 level 3 (tip only)
Animation:
1. Run → legs + tail animation
2. Crawl → body sway animation
3. Stop → freeze
Non-Cube Primitive:
Circles used for ears!
